# V0.4 构筑系统验收标准

## 功能验收

- [x] 坊市生成 3 张商品，商品可连续购买，价格按普通/稀有/传说为 20/35/55 灵石。
- [x] 坊市刷新费用依次为 10/15 灵石，最多刷新 2 次，旧商品不会再次出现为 `ACTIVE`。
- [x] 坊市移除卡牌扣除 30 灵石，每个坊市最多一次；不能移除最后一张有效卡牌。
- [x] `auction` 天关黑市提供一次免费移除入口，也可以跳过。
- [x] 卡牌移除只改变 `run_build_item.status` 为 `REMOVED`，不会物理删除历史记录。
- [x] 战斗加成、流派协同、构筑统计和升级选项只读取 `ACTIVE` 卡牌。
- [x] 剑修、丹修、体修、鬼修均按 2 张基础、3 张强化计算协同。
- [x] 奖励池接收节点、楼层、稀有度、精英数量、事件和当前构筑上下文，并由 seed 保证可复盘。
- [x] 页面显示功法/法宝/符箓数量、四大流派数量、协同效果和当前战斗加成。

## 数据与恢复验收

- [x] `skill_config`、`item_config`、`talisman_config` 可初始化 17 张卡牌。
- [x] 配置初始化幂等，只补缺失数据，不覆盖已有配置。
- [x] `run_shop`、`run_shop_offer` 保存坊市刷新、购买、移除、售罄和关闭状态。
- [x] 刷新页面或调用 GET 存档后，坊市、移除、协同和统计状态保持一致。
- [x] 卡牌快照保留原始名称、稀有度、描述和效果文本，配置变化不影响历史构筑。
